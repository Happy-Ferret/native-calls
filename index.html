Native Calls

Welcome to Native Calls. Here are a few demos to show you it working.

  • End to End tests: The end to end tests show the basic RPC framework and how to use it from JS.
  • Bullet demo: This shows off a full physics simulation using bullet physics engine for C++. All physics is simulated using RPC.
  • Oniguruma This shows off a library that runs regular expression searches using C++, with a JavaScript interface.

Note: You will need to build the demos first. To do this, type `make` to compile and build everything.
