RPCModule Example

Make sure you have built the rpc module, by typing make in this folder.
Open the console and type in: 

EchoRPCModule.Echo("Mohamed", function(result){
    console.log(JSON.stringify(result))
});

How it works

EchoRPCModule is constructed using the RPCModule function. This requires a NaClModule and function definitions. The result is a name-spaced set of functions that you can call directly, as in the example above. 
new RPCModule({
    "module": new NaClModule({src:'../rpc-module.nmf', name:'myRPC', id:"naclRPC", type:'application/x-pnacl'}),
    "functions": [
      {"name": "Echo", "params": ["String"], "returnType": "String"}
    ]
  });
